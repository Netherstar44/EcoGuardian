Desarrolla una aplicación web moderna enfocada en problemáticas ambientales en Colombia.
La plataforma debe ser visualmente atractiva, profesional y completamente responsive (mobile-first), con una paleta de colores basada en tonos verdes (#1B5E20, #2E7D32, #66BB6A, blanco y gris claro).

La versión inicial será Beta, con infraestructura ligera basada en NeonDB para la base de datos y Cloudinary para almacenamiento de imágenes.

🌱 Objetivo del proyecto

Crear una plataforma digital donde los ciudadanos puedan:

Reportar problemas ambientales

Visualizar datos ecológicos de su región

Participar en acciones comunitarias

Acceder a información ambiental

Enfocado inicialmente en Colombia.

🖼️ Almacenamiento de imágenes

Servicio:

Cloudinary

Uso:

subir fotos de reportes ambientales

optimizar imágenes automáticamente

generar URLs CDN

Configuración usando variables de entorno:

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

Flujo de subida:

Usuario sube imagen

Backend recibe archivo

Backend lo envía a Cloudinary

Cloudinary devuelve URL

URL se guarda en NeonDB
